# Design System

## 風格

- 暖色系
- 手帳風
- 日式木質感
- 大圓角
- 柔和陰影
- 手機優先

## 色彩

- Background: `#fff8ec`
- Surface: `#fffdf7`
- Primary: `#8d5a3b`
- Accent: `#d89a56`
- Income: `#5d8c61`
- Expense: `#b76352`

## 字體

- 標題：辰宇落雁體，若未安裝則使用系統書法 / 楷體替代。
- 內文：GenSen Rounded，若未安裝則使用 Noto Sans TC / 系統字體替代。

> 字體檔未內建於此版本，避免授權與檔案大小問題。後續可將合法取得的字體檔放入 `src/assets/fonts/` 並以 `@font-face` 載入。
