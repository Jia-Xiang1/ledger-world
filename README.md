# 記帳天地 Ledger World

暖色手帳風的個人記帳 App。  
目前版本：`v0.2.0-alpha.1`

## 本版內容

- Dashboard 首頁改版
- 新增收入 / 支出
- 編輯與刪除紀錄
- Local Storage 本機儲存
- 今日收入、今日支出、本月結餘
- 最近五筆紀錄
- GitHub Actions 自動部署
- PWA 基礎設定

## 開發

```bash
npm install
npm run dev
npm run build
```

## 部署

推送到 `main` 後，GitHub Actions 會自動部署到 GitHub Pages。
